# Православные веб-ресурсы: Lestvica Orthodox

Мир вам, дорогие братья и сестры!

В этом репозитории я делюсь своими наработками для WordPress темы **Blocksy**, адаптированными под нужды православных приходов и библиотек. 

### Что здесь находится:

**"I"** — Библия для вывода на страницах сайта через `JS`, `Ajax` или любым другим способом (формат `.json`):
* **1.bibliya-syn-77.json**: Файл Библии на русском языке — "SYN-77: Синодальный перевод RST".
* **2.hebrew_bible_WLC.json**: Файл Библии на иврите — "WLC: Westminster Leningrad Codex".
* **3.ASV.json**: Файл Библии на английском языке — "ASV: American Standard Version (1901)".
* **4.GerSch.json**: Файл Библии на немецком языке — "GerSch: Schlachter Bibel (1951)".
* **5.GreVamvas.json**: Файл Библии на греческом языке — "GreVamvas: Neophytos Vamvas's translation into modern Greek (1850)".
* **6.KJV.json**: Файл Библии на английском языке — "KJV: King James Version (1769) с номерами Стронга и морфологией".
* **7.PolUGdanska.json**: Файл Библии на польском языке — "PolUGdanska: Updated Gdańsk Bible".
* **8.UkrOgienko.json**: Файл Библии на украинском языке — "UkrOgienko: Українська Біблія. Переклад Івана Огієнка".
* **9.VulgClementine.json**: Файл Библии на латыни — "VulgClementine: Clementine Vulgate".

**"II"** — Вспомогательные файлы для виджетов, глобальных блоков и настройки контента:
* **10.Bible_Reading_Plan.json**: Файл плана чтения глав Библии на год.
* **11.csslovar.json**: Словарь малопонятных церковнославянских слов, часто используемых в богослужебных книгах.
* **12.otecnik.json**: Сборник цитат святых отцов. Содержит **19 479** мудрых изречений древних и современных подвижников.


### Как использовать:
Скачайте нужный файл и используйте его в коде вашего сайта или импортируйте в соответствующий плагин.

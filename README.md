# Православные веб-ресурсы: Lestvica Orthodox

**Мир вам, дорогие братья и сестры!**

В этом репозитории я делюсь своими наработками для **WordPress**, адаптированными под нужды православных приходов и библиотек. 

### Что здесь находится:

**"I"** — Библия для вывода на страницах сайта:
* **1.bibliya-syn-77.json**: Файл - **Библия на русском** языке — "SYN-77: Синодальный перевод RST".
* **2.hebrew_bible_WLC.json**: Файл - **Библия на иврите** — "WLC: Westminster Leningrad Codex".
* **3.ASV.json**: Файл - **Библия на английском** языке — "ASV: American Standard Version (1901)".
* **4.GerSch.json**: Файл - **Библия на немецком** языке — "GerSch: Schlachter Bibel (1951)".
* **5.GreVamvas.json**: Файл - **Библия на греческом** языке — "GreVamvas: Neophytos Vamvas's translation into modern Greek (1850)".
* **6.KJV.json**: Файл - **Библия на английском** языке — "KJV: King James Version (1769) с номерами Стронга и морфологией".
* **7.PolUGdanska.json**: Файл - **Библия на польском** языке — "PolUGdanska: Updated Gdańsk Bible".
* **8.UkrOgienko.json**: Файл - **Библия на украинском** языке — "UkrOgienko: Українська Біблія. Переклад Івана Огієнка".
* **9.VulgClementine.json**: Файл - **Библия на латыни** — "VulgClementine: Clementine Vulgate".

**"II"** — Вспомогательные файлы для виджетов, глобальных блоков и настройки контента:
* **10.Bible_Reading_Plan.json**: Файл - **План чтения глав Библии на год**.
* **11.csslovar.json**: **Словарь** малопонятных Церковно - Славянских слов, часто используемых в богослужебных книгах.
* **12.otecnik.json**: Сборник цитат Святых Отцов. Содержит **19 479** мудрых изречений древних и современных подвижников.

**"III"** — Нормативные и богослужебные основы:
* **13.dogmats.json**: Полный сборник догматов и постановлений **Святых Соборов** и **Святых Отцов**.
* **14.tipikon.json**: **Типикон**. Устав Богослужения. Все главы кроме Месяцослова.
* **15.troparion.json**: **Полный Тропарион**. "Годовой круг", "Общие", "Триоди" и тд..

**"IV"** — Нормативные и богослужебные основы:
* **16.csfonts.zip**: Папка с **Церковно - Славянскими** (и не только) - Шрифтами. В комплекте 130 штук.


### Как использовать:
Скачайте нужный файл и используйте его в коде вашего сайта через **`JS`**, **`Ajax`** или любым другим способом; или импортируйте в соответствующий плагин.
